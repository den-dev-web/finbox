# FinBox — Dashboard

Одностраничный дашборд аналитики финансовых метрик.

## Быстрый старт

```bash
npm run dev
```

Откройте `http://localhost:5173`.

```bash
npm run build
```

## Структура проекта

```
index.html
src/
  assets/
    fonts/
    icons/
    images/
  js/
  styles/
data/
```

## Примечания
- Шрифты ожидаются в `src/assets/fonts` в формате woff2.
